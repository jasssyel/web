Proyectos

Gestor de contraseñas C++

Desarrollé un gestor de contraseñas en C++ que permite a los usuarios almacenar y organizar sus contraseñas de manera segura. El programa utiliza cifrado para proteger la información y ofrece una interfaz sencilla para agregar, eliminar y buscar contraseñas.

Página web / portafolio web

Creación de esta página web personal utilizando HTML, CSS y JavaScript para mostrar mi portafolio de proyectos y habilidades. La página es responsiva, con un diseño limpio y moderno, e incluye secciones sobre mí, mis proyectos, habilidades y contacto.

Logica digital / VHDL

Desarrollo de circuitos lógicos utilizando el lenguaje VHDL para la implementación de sistemas digitales.
